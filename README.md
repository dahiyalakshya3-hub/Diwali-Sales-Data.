# 📊 Diwali Sales Data Analysis Project

## 🧾 Project Overview
This project analyzes Diwali sales data using Python to understand customer behavior, sales trends, and business insights.

The goal is to explore data and find meaningful patterns that help improve business decisions.

---

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset
The dataset contains Diwali sales information including:
- Customer details
- Gender
- Age group
- State
- Product category
- Purchase amount

---

## 📊 Key Analysis Performed

✔ Gender-wise sales analysis  
✔ Age group-wise spending behavior  
✔ State-wise top customers  
✔ Product category performance  

---

## 🔍 Key Insights

- Female customers contribute more to total sales  
- Age group 26–35 is the highest spending group  
- Uttar Pradesh and Maharashtra are top states in sales  
- Food & Electronics categories are most popular  

---

## 📈 Visualizations

Bar charts and graphs were created using Matplotlib to understand:
- Sales distribution
- Customer behavior
- Top-performing categories
